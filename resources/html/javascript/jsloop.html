Schleifen (Loops)

Eine Schleife ist ein Programmierwerkzeug, das eine Reihe von Anweisungen wiederholt, bis eine festgelegte Bedingung, die sogenannte Abbruchbedingung, erreicht wird. Schleifen werden ständig verwendet! Der allgemeine Begriff "iterieren" bedeutet einfach "wiederholen".

Wenn eine Aufgabe im Code mehrfach ausgeführt werden muss, wird diese oft in einer Funktion gebündelt. Ähnlich wird eine Schleife geschrieben, wenn ein Prozess mehrfach hintereinander wiederholt werden muss. Schleifen ermöglichen die Erstellung effizienter Programme, die Prozesse automatisieren und skalierbar sowie gut verwaltbar sind.

Eine Schleife wiederholt eine Aktion, bis eine bestimmte Bedingung erfüllt ist. Sobald die Bedingung erfüllt ist, stoppt die Schleife und der Computer fährt mit dem nächsten Teil des Programms fort.

Achtung: Unendliche Schleifen entstehen, wenn die Abbruchbedingung nie erfüllt wird, wordurch das Programm blockiert. In solchen Fällen hilft ein Seiten-Reload, um die Ausführung zu stoppen.

Manuelles Wiederholen von Aufgaben

Bevor eigene Schleifen geschrieben werden, soll gezeigt werden, wie umständlich es wäre, wenn eine wiederholte Aufgabe immer wieder manuell getippt werden müsste.

Beispiel: 

const vacationSpots = ['Thailand', 'Brazil', 'Romania'];

console.log(vacationSpots[0]);
console.log(vacationSpots[1]);
console.log(vacationSpots[2]);

Die for - Schleife

Anstatt denselben Code immer wieder zu schreiben, erlauben Schleifen dem Computer, einen bestimmten Codeblock automatisch zu wiederholen. Eine Möglichkeit, dem cComputer diese Anweisungen zu geben, ist die Verwendung von for - Schleife.

Die typische for - Schleife enthält eine Iterationsvariable, die in allen drei Ausdrücken verwendet wird: Initialisierung, Abbruchbedingung und Aktualisierung der Variable bei jeder Iteration. Iterationsvariablen können beliebige Namen haben, es ist aber gute Praxis, beschreibende Namen zu verwenden.

Eine for - Schleife enthält drei durch Semikolons getrennte Ausdrücke in Klammern:

  • Initialisierung:
    Startet die Schleife und deklariert oft die Iterationsvariable.
  • Abbruchbedingung:
    Die Bedingung, die geprüft wird - ist sie wahr, läuft der Codeblock, ist sie falsch, stoppt die Schleife.
  • Iterationsschritt:
    Aktualisiert die Iterationsvariable nach jeder Runde. 
for (let counter = 0; counter < 4; counter++) {
  console.log(counter);
}
/*
Ausgabe:
0
1
2
3
*/

Erläuterung:

  • Die Initialisierung ist let counter = 0, also beginnt das Zählen bei 0.
  • Die Abbruchbedingung counter < 4 sorgt dafür, dass die Schleife läuft, solange counter kleiner als 4 ist.
  • Der Iterationsschnitt counter++ erhöht counter nach jeder Wiederholung um 1.
  • Der Codeblock gibt den aktuellen Wert von counter aus und lä$uft bis zur Abbruchbedingung.

Beispiel:

for (let counter = 5; counter < 11; counter++) {
  console.log(counter);
}
/*
Ausgabe:
5
6
7
8
9
10
*/

Schleife rückwärts durchlaufen

Soll eine for - Schleife rückwärts zählen, muss die Iterationsvariable in der Initialisierung auf dem höchsten Wert gesetzt werden, die Abbruchbedingung prüft,

ob der Wert grösser oder gleich dem niedrigsten Wert ist, und die Iterationsvariable wird jeweils verringert.

Beispiel:

for (let counter = 3; counter >= 0; counter--) {
  console.log(counter);
}

Schleifen durch Arrays

for - Schleifen sind praktisch, um durch Datenstrukturen wie Arrays zu iterieren. Arrays speichern geordnete Daten wie Kunden- oder Produktinformationen. Um z.B. jeden Preis in einen Produktkatalog zu erhöhen, kann eine for - Schleife alle Elemente durchlaufen.

Typischerweise wird in der Abbruchbedingung die Eigenschaft .length des Arrays genutzt, um die Schleife über alle Elemente laufen zu lassen.

Beispiel:

const animals = ['Grizzly Bear', 'Sloth', 'Sea Lion'];
for (let i = 0; i < animals.length; i++) {
  console.log(animals[i]);
}
/*
Ausgabe:
Grizzly Bear
Sloth
Sea Lion
*/

Die Iterationsvariable i steht oft für "index". Die Bedingung i < animals.length sorgt dafür, dass nicht über das letzte Element hinausgegangen wird, da Arrays bei 0 beginnen.

Beispiel: 

const vacationSpots = ['Bali', 'Paris', 'Tulum'];

for (let i = 0; i < vacationSpots.length; i++) {
  console.log('I would love to visit ' + vacationSpots[i]);
}

Verschachtelte Schleifen

Eine Schleife innerhalb einer anderen Schleife heisst verschachtelte Schleife. Ein Anwendungsfall ist der Vergleich von Elementen zweier Arrays. Für jede Runde der äusseren Schleife läuft die innere Schleife komplett durch.

Beispiel: 

const arrayA = [6, 19, 20];
const arrayB = [19, 81, 2];

for (let i = 0; i < arrayA.length; i++) {
  for (let j = 0; j < arrayB.length; j++) {
    if (arrayA[i] === arrayB[j]) {
      console.log('Both arrays have the number: ' + arrayB[j]);
    }
  }
}

Die while - Schleife

Eine andere Schleifenart ist die while - Schleife. Dabei wird eine Bedingung geprüft und der Codeblock so lange ausgeführt, wie die Bedingung wahr ist.
br Beispiel Umwandlung einer for- in eine while-Schleife: 

// for-Schleife
for (let counterOne = 1; counterOne < 4; counterOne++) {
  console.log(counterOne);
}

// while-Schleife
let counterTwo = 1;
while (counterTwo < 4) {
  console.log(counterTwo);
  counterTwo++;
}

Wichtig: Die Iterationsvariable muss innerhalb der Schleife aktualisiert werden, sonst entsteht eine unendliche Schleife.

while - Schleifen eignen sich, wenn nicht bekannt ist, wie oft die Schleife laufen muss, z.B. "iss, solange Hunger besteht".

Beispiel: 

const cards = ['diamond', 'spade', 'heart', 'club'];
let currentCard;

while (currentCard !== 'spade') {
  currentCard = cards[Math.floor(Math.random() * 4)];
  console.log(currentCard);
}

Do...while - Schleife

Der Codeblock wird mindestens einmal ausgeführt und danach die Abbruchbedingung geprüft. Falls die Bedingung wahr ist, läuft die Schleife weiter.

Syntax: 

let countString = '';
let i = 0;

do {
  countString += i;
  i++;
} while (i < 5);

console.log(countString);

Unterschied zur while - Schleife: do...while führt den Codeblock mindestens einmal aus, unabhängig von der Bedingung.

Das break - Schlüsselwort

Mit break kann eine Schleife vorzeitig verlassen werden, auch wenn die Abbruchbedingung noch nicht erfüllt ist.

Beispiel: 

for (let i = 0; i < 99; i++) {
  if (i > 2) {
    break;
  }
  console.log('Banana.');
}

console.log('Orange you glad I broke out the loop!');
/*
Ausgabe.
Banana.
Banana.
Banana.
Orange you glad i broke out the loop!
*/

break ist besonders nützlich bei der Arbeit mit grossen Datenstrukturen, um zusätzliche Abbruchbedingungen einzubauen.

Zusammenfassung

  • Schleifen automatisieren wiederholende Aufgaben, um manuelles Programmieren zu vermeiden.
  • for - Schleifen verwenden eine Iterationsvariable, die erhöht oder verringert wird.
  • for - Schleifen eignen sich gut für Arrays.
  • Verschachtelte Schleifen sind Schleifen innerhalb von Schleifen.
  • while - Schleifen ermöglichen flexible Abbruchbedingungen.
  • Abbruchbedingungen sind wichtig, um unendliche Schleifen zu verhindern.
  • Do...while - Schleifen fürhen den Code mindestens einmal aus.
  • break ermöglicht das vorzeitige Verlassen einer Schleife.
