Iteratoren

Stell dir vor, es gibt eine Einkaufsliste und es soll geprüft werden, was sich auf dieser Liste befindet - dazu könnte jede Zeile durchgegangen und jedes Element abgelesen werden. Diese häufige Aufgabe ähnelt der Iteration über ein Array.

Ein Werkzeug, das zur Verfügung steht, ist die for - Schleife. Zusätzlich gibt es eingebaute Array - Methoden, die spezialisierte Iterationen bieten.

Die eingebauten JavaScript - Araay - Methoden, die beim Iterieren helfen, nennt man Iterationsmethoden, oft auch als Iteratoren bezeichnet. Iteratoren sind Methoden, die auf Arrays angewendet werden, um Elemente zu manipulieren und Werte zurückzugeben.

Die .forEach() - Methode

Die erste Iterationsmethode, die behandelt wird, ist .forEach(). Der Name deutet darauf hin, dass .forEach() denselben Code für jedes Element eines Arrays ausführt

Beispiel: 

const groceries = ['apple', 'banana', 'carrot', 'date'];

groceries.forEach(item => console.log(item));

Erläuterung der Syntax von .forEach():

  • groceries.forEach() ruft die Methode forEach für das Array groceries auf.
  • .forEach() benötigt eine Callback - Funktion als Argument. Eine Callback - Funktion ist eine Funktion, die als Argument an eine andere Funktion übergeben wird.
  • .forEach() durchläuft das Array und führt für jedes Element die Callback - Funktion aus. Dabei wird das aktuelle Element als Argument an die Callback - Funktion übergeben.
  • Der Rückgabewert von .forEach() ist immer undefined.

Ein alternativer Weg, die Callback - Funktion zu übergeben, ist die Verwendung von Pfeilfunktionen: 

groceries.forEach(item => console.log(item));

Die .map() - Methode

Die zweite Iterator - Methode ist .map(). Wenn .map() auf ein Array angewendet wird, erwartet es eine Callback - Funktion als Argument und gibt ein neues Array zurück. 

const numbers = [1, 2, 3, 4, 5];

const bigNumbers = numbers.map(number => number * 10);
console.log(bigNumbers); // Ausgabe: [10, 20, 30, 40, 50]

Erläuterung:

  • .map() funktioniert ähnlich wie .forEach(), der grosse Unterschied ist, dass .map() ein neues Array zurückgibt.
  • In diesem Beispiel werden die Zahlen im Array numbers mit 10 multipliziert und in das neue Array bigNumbers gespeichert.

Die .filter() - Methode

Eine weitere nützliche Iterator - Methode ist .filter(). Diese Methode gibt ein neues Array zurück, das nur die Elemente enthält, die eine bestimmte Bedingung erfüllen. 

const words = ['chair', 'music', 'pillow', 'brick', 'pen', 'door'];

const shortWords = words.filter(word => word.length < 6);
console.log(shortWords); // Ausgabe: ['chair', 'music', 'brick', 'pen', 'door']

Erläuterung:

  • .filter() überprüft jedes Element des Arrays, um festzustellen, ob es eine Bedingung erfüllt (hier, ob die Wortlänge kleiner als 6 ist).
  • Elemente, die die Bedingung erfüllen, werden in das neue Array aufgenommen.

Die .findIndex() - Methode

Manchmal muss der Index eines Elements in einem Array gefunden werden. Die Methode .findIndex() gibt den Index des ersten Elements zurück, das in der Callback - Funktion als true bewertet wird. 

const jumbledNums = [123, 25, 78, 5, 9];

const lessThanTen = jumbledNums.findIndex(num => num < 10);
console.log(lessThanTen); // Ausgabe: 3
console.log(jumbledNums[3]); // Ausgabe: 5

Erläuterung:

  • .findIndex() gibt den Index des ersten Elements zurück, das die angegebene Bedingung erfüllt. Wird keine Übereinstimmung gefunden, wird -1 zurückgegeben.

Die .reduce() - Methode

Die Methode .reduce() reduziert das Array auf einen einzelnen Wert, indem sie jeden Wert des Arrays mit einem "Accumulator" zusammenfasst. 

const numbers = [1, 2, 4, 10];

const summedNums = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
console.log(summedNums); // Ausgabe: 17

Erläuterung:

  • .reduce() nimmt einen "Accumulator" und einen "currentValue". Die Rückgabewerte werden in accumulator gespeichert, der im nächsten Iterationsschritt wiederverwendet wird.
  • Optional kann der Accumulator - Wert am Anfang festgelet werden.
