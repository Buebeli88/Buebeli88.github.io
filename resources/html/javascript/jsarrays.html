Arrays

Das Organisieren und Speichern von Daten ist ein grundlegendes Konzept in der Programmiereung.

Eine Methode zur Organisation von Daten im Alltag ist das Erstellen von Listen.

Beispiel:

Neujahrvorsätze:

  1. Ein Tagebuch fürhen
  2. Einen Falkenerei-Kurzs besuchen
  3. Jonglieren lernen

Diese liste in JavaScript als Array:

let neujahrsvorsätze = ['Ein Tagebuch führen', 'Einen Falkenerei-Kurzs besuchen', 'Jonglieren lernen'];

Ein Array erstellen

Ein Array kann durch ein sogenanntes Array-Literal erstellt werden. Ein Array-Literal entsteht durch das Einschliessen der Elemente in eckige Klammern []. Arrays können entweder gleichartige oder unterschiedliche Datentypen enthalten.

Diagramm:
Ein Array-Literal mit deri Elementen (ein String, eine Zahl, ein Boolean), durch Kommas getrennt und von eckigen Klammern umbegen.

Syntaxanalyse:

  • Das Array wird duchr eckige Klammern [ ] dargestellt.
  • Jedes enthaltene Element wird als Element bezeichnet.
  • Drei unterschiedliche Elemente befinden sich in array.
  • Jedes Element hat einen unterschiedlichen Datentyp.

Ein Array kann auch einer Variablen zugewiesen werden - wie im vorherigen Beispiel: 

let neujahrsvorsätze = ['Ein Tagebuch führen', 'Einen Falkenerei-Kurs besuchen', 'Jonglieren lernen'];

Elemente zugreifen

Jese Array-Element besitzt eine nummerierte Position - den Index. Über diesen Index kann ein einzelnes Element angesprochen werden. JavaScript-Arrays sind nullbasiert, d.h. das Zählen beginnt bei 0:

Beispiel: 

let städte = ['New York', 'Berlin', 'Tokio'];
console.log(städte[0]);
// Ausgabe: 'New York'

Erklärung:

  • Das Array städte enthält drei Elemente.
  • Über Klammernotation [ ] und Index erfolgt der Zugriff.
  • städte[0] greift auf das erste Element ('New York') zu.

Auch einzelne Zeichen in einem String sind über Indizes zugänglich: 

const hallo = 'Hallo Welt';
console.log(hallo[6]);
// Ausgabe: W

Arrays mit let und const

Variablen können mit let oder const deklariert werden. Mit let deklarierte Variablen sind neu zuweisbar. Mit const deklarierte Variablen nicht.

Ein Array, das mit const deklariert wurde, bleibt änderbar - der Inhalt kann verändert werden, aber die Referenz nicht.

Beispielhafte Arrays: gewürze und utensilien 

let condiments = ['Ketchup', 'Mustard', 'Soy Sauce', 'Sriracha'];
const utensils = ['Fork', 'Knife', 'Chopsticks', 'Spork'];

condiments[0] = 'Mayo';
console.log(condiments);
// Ausgabe: ['Mayo', 'Mustard', 'Soy Sauce', 'Sriracha']
condiments = ['Mayo'];
console.log(condiments);
// Ausgabe: ['Mayo']
utensils[3] = 'Spoon';
console.log(utensils);
// Ausgabe: ['Fork', 'Knife', 'Chopsticks', 'Spoon']

Die .length - Eigenschaft

Die .length - Eigenschaft gibt die Anzahl der Elemente im Array zurück: 

const neujahrsvorsätze = ['Ein Tagebuch führen', 'Einen Falknerei-Kurs besuchen'];

console.log(neujahrsvorsätze.length);
// Ausgabe: 2

Zugriff über Punktnotation: array.length

Die .push() - Methode

.push() fügt ein oder mehrere Elemente am Ende eines Arrays hinzu: 

const artikelVerfolgung = ['Artikel 0', 'Artikel 1', 'Artikel 2'];

artikelVerfolgung.push('Artikel 3', 'Artikel 4');

console.log(artikelVerfolgung); 
// Ausgabe: ['Artikel 0', 'Artikel 1', 'Artikel 2', 'Artikel 3', 'Artikel 4']
  • .push() mutiert das Array.
  • Funktioniert mit einem oder mehreren Argumenten.
  • Wird über Punktnotation aufgerufen.

Die .pop() - Methode

.pop() entfernt das letzte Element eines Arrays und gibt dieses zurück: 

const artikelVerfolgungNeu = ['Artikel 0', 'Artikel 1', 'Artikel 2'];

const entfernt = artikelVerfolgungNeu.pop();

console.log(artikelVerfolgungNeu); 
// Ausgabe: ['Artikel 0', 'Artikel 1']
console.log(entfernt);
// Ausgabe: Artikel 2
  • .pop() benötigt keine Argumente.
  • Gibt das entfernte Element zurück.
  • Mutiert das Array.

Weitere Array-Methoden

Zusätzlich zu .push() und .pop() gibt es viele weitere Array-Methoden.

Beispiel:

  • .join()
  • .slice()
  • .splice()
  • .shift()
  • .unshift()
  • .concat()

Einige Methoden mutieren das Array (z.B. .push(), .pop() ), andere nicht. Details in der Dokumentation (z.B. MDM Web Docs).

Arrays und Funktionen

Wird ein Array an eine Funktion übergeben und innerhalb der Funktion geänder, bleibt diese Änderung bestehen - sogenanntes Pass-by-Reference: 

const blumen = ['Pfingstrose', 'Narzisse', 'Ringelblume'];

function blumeHinzufügen(arr) {
  arr.push('Lilie');
}

blumeHinzufügen(blumen);
console.log(blumen);
// Ausgabe: ['Pfingstrose', 'Narzisse', 'Ringelblume', 'Lilie']

Beispiel mit Muatation und Entfernen:

const konzepte = ['Arrays', 'können', 'verändert', 'werden'];

function ändereArray(arr) {
  arr[3] = 'VERÄNDERT';
}
ändereArray(konzepte);
console.log(konzepte);

function entferneElement(arr) {
  arr.pop();
}
entferneElement(konzepte);
console.log(konzepte);

Verschachtelte Arrays

Ein verschateltes Array enthält weitere Arrays.

Beispiel: 

const verschachteltesArray = [[1], [2, 3]];

console.log(verschachteltesArray[1]); // Ausgabe: [2, 3]
console.log(verschachteltesArray[1][0]); // Ausgabe: 2

Mehrere Ebenen von Klammernotation ermöglicheen den Zugriff auf tiefer liegende Werte.

Beispiel: 

const zahlenCluster = [[1, 2], [3, 4], [5, 6]];

const zielwert = zahlenCluster[2][1];
console.log(zielwert); // Ausgabe: 6

Zusammenfassung

Die wichtigsten Konzepte:

  • Arrays sind geordnete Listen zur Datenspeicherung in JavaScript.
  • Erstellung mit eckigen Klammern [ ].
  • Elemente haben nummerierte Positionen (Index), beginnend bei 0.
  • Zugriff auf Elemente über den Index: array[0].
  • Änderung eines Elements: array[0] = 'neuer Wert'.
  • .length gibt die Anzahl der Elemente zurück.
  • Methoden wie .push() und .pop() fügen Elementen hinzu oder entfernen sie.
  • Weitere Methoden: .slice(), .shift(), .splice(), .join() etc.
  • Manche Methoden mutieren das Arra, andere nicht.
  • Arrays bleiben auch bei const deklarierbar veränderbar (nicht nue zuweisbar).
  • Veränderungen in Arrays innerhalb von Funktionen wirken sich auch ausserhalb von aus(Pass-by-Reference).
  • Arrays können andere Arrays enthalten (verschachtelte Arrays).
  • Zugriff auf verschachtelte Elemente durch Ketten von Indizers.

Die Fähigkeit, Arrays zu manipulieren, ist essenziell beim Umgang mit Datenstrukturen in JavaScript.
