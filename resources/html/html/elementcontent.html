Element Content

Der Begriff "Element Content" in HTML bezieht sich auf den Inhalt, der zwischen dem öffnenden und schließenden Tag eines HTML-Elements steht.

Dieser Inhalt kann reiner Text, andere HTML-Elemente oder sogar andere Arten von Daten sein, die innerhalb des Elementbereichs definiert sind.

Opening Tag

Ein HTML-Opening-Tag (auch Start-Tag genannt) ist der Beginn eines HTML-Elements und kennzeichnet den Beginn eines Abschnitts,

der von diesem Element umschlossen wird. Es besteht aus dem Elementnamen in spitzen Klammern.

Beispiel:

 <div>

Closing Tag

Ein Closing Tag (oder End-Tag) in HTML kennzeichnet das Ende eines HTML-Elements.

Es besteht aus einem Schrägstrich (/) gefolgt vom Elementnamen, eingeschlossen in spitze Klammern.

Beispiel:

</div>

Attribute Name and Values

In HTML beziehen sich Name und Wert auf die Attribute eines HTML-Elements. Ein Attribut besteht immer aus einem Namen (Name) und einem zugehörigen Wert (Value),

die zusammen eine Erweiterung oder zusätzliche Information zum Element darstellen. Man schreibt sie im öffnenden Tag des Elements.

Beispiel:

<Elementname Name ="Wert">Inhalt</Elementname>
