HTML Tables

In HTML ist ein <table> Element eine Struktur, die zur Darstellung von tabellarischen Daten verwendet wird. Es besteht aus Zeilen (<tr>) und Spalten (<td> für Datenzellen oder <th> für Kopfzellen), die gemeinsam ein Raster bilden. Tabellen können für verschiedene Zwecke eingesetzt werden, wie z.B. die Darstellung von Informationen in Listen, Preisvergleichen oder Statistiken.

<table> Table Element

Im HTML-Dokument wird ein <table> Element benutzt um eine Tabelle mit Reihen (rows) und Spalten (columns) zu erstellen.

Beispiel:

Code:

<table>
  <!-- rows and columns will go here -->
</table>

<thead> Table Head Element

Das <thead> Element ist der "header" der Tabelle.

Beispiel:

Code:

<table>
  <thead>
    <tf>
      <th>heading 1</th>
      <th>heading 2</th>
    </tr>
  </thead>
</table>

HTML (Border with CSS):

heading 1 heading 2

<tbody> Table Body Element

Das <tbody> Element ist ein Semantisches Element. Es beinhaltet die ganzen Tabellendaten ausser der "header" und "footer" Daten. Der <tbody> enthält die <tr> "row" Elemente und zeigt an dass das <tr> Element den "body" der Tabelle ausmachen. Ein <table> Element kann nicht <tbody> UND <tr> als direkte "Kinder" haben.

Beispiel:

Code:

<table>
  <tbody>
    <tr>
      <td>row 1</td>
    </tr>
    <tr>
      <td>row 2</td>
    </tr>
  </tbody>
</table>

row 2

<tfoot> Table Footer Element

Das <tfoot> Element gibt der Tabelle ein "footer" Reihe, welche z.B. zum zusammenfassen einer Reihe genutzt werden kann.

Beispiel:

<table>
  <thead>
    ...
  </thead>
  <tbody>
    ...
  </tbody>
  <tfoot>
    ...
  </tfoot>
</table>

<tr> Table Row Element

Das <tr> Element wird benutzt, um der Tabelle eine Zusätzliche Reihe für Daten <td> oder Headings <th> zu geben.

Beispiel:

Code:

<table>
  <tr>
    ...
  </tr>
</table>

<th> Table Heading Element

Das <th> Element wird benutzt um einer Reihe oder Spalte einen Titel zu verpassen. Dieses Element muss in einem <tr> Element sein.

Beispiel:

Code:

<table>
  <tr>
    <th>column</th>
  </tr>
  <tr>
    <td>1</td>
  </tr>
</table>

HTML (Border with CSS):

column
1

<td> Table Data Element

Das <td> Element wird in einer Reihe (<tr>) eingesetzt um Daten in der Tabelle anzuzeigen.

Beispiel:

Code:

<table>
  <tr>
    <td>cell one data</td>
    <td>cell two data</td>
  </tr>
</table>

HTML (Border with CSS):

cell one data cell two data

"rowspan" Attribut

Ähnlich zum "colspan" Attribut wird das "rowspan" Attribut verwendet, um einem Element zu sagen, es solle mehrere Reihen in der Tabelle besetzten. Die "value" des Attributs geht von 1 (default) bis hin zu 65534.

Beispiel:

Code:

<table>
  <tr>
    <th>row 1:</th>
    <td>col 1</td>
    <td>col 2</td>
  </tr>
  <tr>
    <th rowspan="2"> (this row will span 2 rows):</th>
    <td>col 1</td>
    <td>col 2</td>
  </tr>
  <tr>
    <td>col 1</td>
    <td>col 2</td>
  </tr>
  <tr>
    <th>row 3:</th>
    <td>col 1</td>
    <td>col 2</td>
  </tr>
</table>

HTML (Border with CSS):

row 1: col 1 col 2
(this row will span 2 rows): col 1 col 2
col 1 col 2
row 3: col 1 col 2

"colspan" Attribut

Das "colspan" Attribut in einem Header <th> oder in Tabellendaten <td> gibt an wieviele Reihen in einer Spalte benutzt werden. Die «value» geht von 1(default) bis hin zu 1000.

Beispiel:

Code:

<table>
  <tr>
    <th>row 1:</th>
    <td>col 1</td>
    <td>col 2</td>
    <td>col 3</td>
  </tr>
  <tr>
    <th>row 2:</th>
    <td colspan="2">col 1 (will span 2 columns)</td>
    <td>col 2</td>
    <td>col 3</td>
  </tr>
</table>

HTML (Border with CSS):

row 1: col 1 col 2 col 3
row 2: col 1 (will span 2 columns) col 2 col 3
