Command Lines

Eine Command Line, auch Kommandozeile, Konsole oder Terminal genannt, ist eine textbasierte Schnittstelle, über die ein Benutzer mit einem Computer interagiert, indem er Befehle eingibt.

Anstatt eine grafische Benutzeroberfläche (GUI) zu verwenden, werden Befehle direkt in die Zeile eingegeben und vom System ausgeführt.

Filesystem Struktur

Das Dateisystem eines Computers organisiert die vom Computer gespeicherten Daten so, dass sie vom Benutzer leicht abgerufen werden können. Dateien werden in der Regel in einer baumartigen Struktur dargestellt,

bei der jedes übergeordnete Verzeichnis eine beliebige Anzahl von untergeordneten Verzeichnissen enthalten kann. Das Wurzelverzeichnis (Root-Verzeichnis) befindet sich dabei an der Basis des Baums.

The Command Line

Die Kommandozeile ermöglicht es einem Benutzer, im Dateisystem zu navigieren und eingebaute Programme oder eigene Skripte auszuführen.

In Unix wird die Kommandozeilenschnittstelle Bash genannt, und die Eingabeaufforderung (Shell Prompt) ist das $.

Beispiel:

$

Print Working Directory pwd

Der Shell-Befehl `pwd` zeigt den Dateipfad vom Root-Verzeichnis bis zum aktuellen Arbeitsverzeichnis an.

Beispiel:

$ pwd
/Users/sonny/Downloads

List ls

Der Shell-Befehl ls wird verwendet, um den Inhalt eines Verzeichnisses aufzulisten. Wenn keine Argumente angegeben werden, listet er den Inhalt des aktuellen Arbeitsverzeichnisses auf.

Beispiel:

$ ls
Desktop resume.pdf photo.png

Change Directory cd

Der Shell-Befehl cd wird verwendet, um sich im Dateisystem eines Computers zu bewegen.

Er akzeptiert verschiedene Argumente:

  • Vollständige Dateipfade.
  • Name von Unterverzeichnissen des aktuellen Verzeichnisses.
  • .. das übergeorgnete Verzeichnis des aktuellen Verzeichnisses.

Beispiel:

$ cd some-directory
$ cd ../some-directory

Make Directory mkdir

Der Shell-Befehl `mkdir` wird verwendet, um ein neues Verzeichnis im Dateisystem entsprechend seinem Argument zu erstellen. Wird ein Dateipfad angegeben,

wird das neue Verzeichnis am Ende dieses Pfads angelegt. Andernfalls wird das neue Verzeichnis im aktuellen Arbeitsverzeichnis erstellt.

Beispiel:

$ mkdir new-directory
$ ls
old-directory new-directory

Create New File touch

Der Shell-Befehl touch erstellt eine neue Datei im aktuellen Arbeitsverzeichnis mit dem angegebenen Namen.

Beispiel:

$ touch grocery-list.txt

Helper Commands

Hilfsbefehle für die Kommandozeile sind unter anderem:

  • clear, um das Terminal zu löschen
  • tab, um die Eingabe automatisch zu vervollständigen
  • ↑ und ↓, um durch vorherige Befehle zu blättern
