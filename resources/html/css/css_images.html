Bilder

Hintergrundbild

Die CSS-Eigenschaft background-image legt das Hintergrundbild eines Elements fest. Dabei wird die Bild-URL im Format url("bildname.jpg") angegeben.

Beispiel CSS

background-image: url("cat-space.gif");

Beispiel mit dem Hintergrundbild

Grösse des Hintergrundbildes

Mit der CSS-Eigenschaft background-size lässt sich die Grösse des hintergrundbildes bestimmen. Der Wert cover bewirkt, dass das Bild den gesamten Hintergrundbereich des Containers abdeckt.

Dabei bleibt das Seitenverhältnis erhalten, sodass Teile des Bildes abgeschnitten werden können, wenn das Bild grösser als der Container ist.

Beispiel:

background-size: cover;

Bild zentrieren

Das Zentrieren eines Bildes mit CSS kann auf verschiedene Arten erfolgen, sowohl horizontal als auch vertikal.

Horizontales Zentrieren

Drei gängige Methoden zum horizontalen Zentrieren von Bildern:

1. margin: auto
Diese Methode funktioniert nur, wenn das Bild als Block-Element definiert wird (Display: block) und eine feste Breite besitzt (nicht 100%).


Dadurch wird der linke und rechte Aussenabstand automatisch ausgeglichen.

Beispiel:

img {
  display: block;
  margin: auto;
  width: 50%;
}

2. text-align: center
Funktioniert, wenn sich das Bild innerhalb eines Block-Containers (z.b. <div>) befindet.

Beispiel:

div {
  text-align: center;
}

3. display: flex
Der Container wird mit display: flex definiert und durch wird das Bild horizontal zentriert.


Das Bild muss eine kleinere Breite als der Container haben, um tatsächlich zentriert zu werden.

Beispie:

div {
  display: flex;
  justify-content: center;
}

img {
  width: 50%;
}

Vertikales Zentrieren

Das vertikale Zentrieren von Bildern gelingt ebenfalls mit display: flex. Neben justify-content: center muss zursätzlich align-items: center im Container definiert werden.

Die Höhe des Containers und die Bildhöhe müssen berücksichtigt werden.

Beispiel:

div {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 1000px;
}

img {
  width: 50%;
  height: 600px;
}

Text zu Bildern in HTML hinzufügen

Text kann auf verschiedene Weise mit Bildern kombiniert werden:

1. Text unter dem Bild:
Bild mit <img>-Tag und Text in einem <p>-Element direkt darunter. 

<img src="../../images/ultramarine1.jpg" alt="Ein Ultramarine" width="300"">

Dies ist ein Ultramarine!

Beispiel:

Dies ist ein Ultramarine!

2. Text über dem Bild (overlay)
Ein Container-Element mit position: relative enthält das Bild und den Text. Der text wird mit position: absolute über das Bild gelegt und zentriert. 

<div class="image-container">
  <img src="../../images/tyranid1.jpg" alt="Ein Tyranid" class="image">
  <div class="overlay-text">Dies ist ein Tyranid!</div>
</div>
.image-container {
  position: relative;
  display: inline-block;
}

.image {
  display: block;
}

.overlay-text {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  color: white;
  background-color: rgba(0, 0, 0, 0.5);
  padding: 10px;
}

Beispiel:

Dies ist ein Tyranid!

3. Bild und Text nebeneinander (mit Flexbox)
Wenn Bild und Text nebeneinander angezeigt werden sollen, wird display: flex verwendet.

HTML:

<div class="container">
 <img src="dein_bild.jpg" alt="Beschreibung" class="image">
 <p class="text">Hier ist der Text neben dem Bild.</p>
</div>

CSS:

.container {
  display: flex;
  align-items: center;
}

.image {
  margin-right: 20px;
}

Beispiel:

Dies ist ein Necron!

Barrierefreiheit: alt-Attribut

Das <img>-Tag sollte immer ein alt- Attribut enthalten, das den Bildinhalt beschreibt - wichtig für Screenreader und bei Ladeproblemen.

Zusätzlicher Tipp: Inline vs. Block-Elemente

Bilder sind standardmässig inline-Elemente, daher funktionieren manche Layout-Eigenschaften (wie margin: auto) nur, wenn das Bild mit display: block behandelt wird.
