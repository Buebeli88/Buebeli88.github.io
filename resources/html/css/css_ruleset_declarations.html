CSS

Cascading Style Sheets (CSS) is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

CSS Ruleset & Declaration

Ein CSS Ruleset 1 oder mehrere "Selectors" und 1 oder mehrerer "Declarations". Eine "Declaration" ist das "Key-Value" pärchen (Key = Property). Eine "Declaration" wird benutzt um die Individuellen-

und Gruppenelementen zu "stylen" und ihnen eine Struktur zu verpassen."Key" und "Value" werden durch ein ":" getrennt und die ganze "Declaration" wird mit einem ";" beendet.

Hier sind ein paar Selektoren

  • Element: <h1>, <p> oder <body>
  • Class: .navigation oder .image wolf
  • ID: #navigation oder #image wolf

Ruleset Format

selector {
property-name: value;
}

CSS Declaration Format
property-name: value;

Property-name (Key) and Value

Beispiel:

h1 is the Selector

h1 {
  text-align: center;
  color: purple;
  width: 100px;
}

HTML & CSS:

h1 is the selector

CSS Comments

Wie in einem HTML-Dokument können wir auch hier, Kommentare einfügen. Diese werden weder eine Funktion ausführen noch im Browser sichtbar sein. Üblich zum Dokumentieren des Codes,

eine Idee erklärung oder zum festhalten von zukünftigen Ideen. Es sollte vermieden werden die Kommentare zur "Sektionierung" zu verwenden.

Beispiel:

/* Dieser Teil wird für die Navigation verwendet */

!important Rule

!important wird in einer Deklaration verwendet um andere Deklarationen für eine "Property zu überschreiben und ignoriert spezifische Selektoren. Es sollte aber generell vermieden werden die !important Rule zu verwenden.

Resource URL's

Die "URL ()" Funktion wird genutzt um Resourcen anzugeben, ähnlich dem File Path in HTML. Diese Funktion wird für diverse "Properties" wie z.B. "background-image" verwendet.

Beispiel:

background-image: url("../resources/image/image.png")
