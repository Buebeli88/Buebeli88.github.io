CSS Colors

Foreground Color

Dies setzt die Textfarbe auf den gewünschten Wert. Die Farbe kann entweder durch über "Keyword", Hexadecimal, rgb oder HSL gesetzt werden.

Beispiel:

Code:

p {
  color: #2a2aff;
}

/*oder*/

span {
  color: green;
}

Background Color

Dies wird benutzt um die Hintergrundfarbe zu setzen.

Beispiel:

Code:

p {
  background-color: blue;
}

/*oder*/

p {
  background-color: blue;
}

Beispiel:

Code:

Hexadecimal Colors

Farben können auch hexidecimal angegeben werden. Die Werte können in 16 verschiedenen "Symbolen" angegeben werden.

0-9 und a-f. Die Farbe besteht aus 3 2er Gruppen, welche für rot, grün und blau stehen.

Beispiel:

.red {
  color: #ff0000;
}
.short-blue {
  color: #00f;
}

Opacity

Dieses Element wird benutzt um die "Durchsichtigkeit" des Elements zu setzten. Die reichweite der Einheit geht von 0 (durchsichtig) bis zu 1 (Opaque/volle Sicht).

Beispiel:

Code:

p {
  opacity: 0.5;
}

rgb() Colors

Farben können mit RBG-Farben angegeben werden, mit der benutzung des "rgb()" Syntax. Die Einheit wird mit 3 "Values" rot, grün und blau repräsentierend gesetzt. Die Grösse geht von 0 zu 255.

Beispiel:

Code:

.hot-pink {
  color: rgb(249, 2, 171);
}

/*oder*/

.green{
  color: rgb(0, 255, 0);
}

HSL Colors

Farben können auch mit dem HSL Color System deklariert werden durch den "hsl()" Syntax. Dieser benutzt 3 Einheiten: "hue" (Die Farbe selbst), "saturation"

(Die Intensität) und "lightness". Die "hue" Einheit wird mit einer Reichweite von 0-360 angegeben, wärend "saturation" und "lightness" in Prozenten angegeben wird.

Beispiel:

Code:

.light-blue {
  background-color: hsl(200, 70%, 50%);
}

HTML:

Dieser Text hat eine Hintergrundfarbe.

Color Alpha Value

Der Alpha Wert bestimmt die Durchsichtigkeit von Farbein in CSS. Dieser Wert kann für RGB und HSL Farben benutz werden, mit dem gebrauch von rgba() und hsla() und gibt beiden Einheiten einen 4ten Wert. Alpha Werte können von 0.0 (komplett Durchsichtig) und 1.0 (volle Sicht) varieren. Es kann auch also "transparent" benutzt werden um den gleichen Effekt wie 0.0 zu erziehlen.

Beispiel:

Code:

.midground {
  background-color: rgba(0, 255, 0, 0.5);
}

/*oder*/

.foreground{
  background-color: hsla(34, 100%, 50%, 0.1);
}

/*oder*/

.transparent{
  color: transparent;
}
